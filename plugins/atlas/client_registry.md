## HOW TO READ THIS FILE

# \- email: used to match Fireflies meeting participants to the correct client

# \- trello\_board: exact board name in Trello (ATLAS will fuzzy-match if slightly off)

# \- programme\_start: date the 12-week programme began (YYYY-MM-DD). ATLAS auto-calculates the current week from this. Use "pending" if not yet started.

# \- team: team members assigned to this client

# \- language: primary language for client-facing communication

# \- tone\_profile: auto-populated by ATLAS from Fireflies transcripts — do not edit manually
